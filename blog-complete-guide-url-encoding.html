Encoding in Different Programming Languages

JavaScript

JavaScript gives you two main functions: encodeURIComponent and encodeURI. The first one is best for encoding query parameters, while the second handles full URLs but leaves some characters untouched. 

// encodeURIComponent - encodes almost everything
const encoded = encodeURIComponent("Hello World! &?=+");
console.log(encoded); // Hello%20World!%20%26%3F%3D%2B

// encodeURI - better for whole URLs
const encodedURI = encodeURI("https://example.com/hello world");
console.log(encodedURI); // https://example.com/hello%20world

// Decoding
const decoded = decodeURIComponent(encoded);
console.log(decoded); // Hello World! &?=+

Warning: Don‚Äôt use encodeURI() for query parameters‚Äî it skips &, ?, and =, which can break your link. Stick with encodeURIComponent() for those.

Python

In Python, the urllib.parse module has what you need. quote and quote_plus are the main players: 

from urllib.parse import quote, quote_plus, unquote

# Basic encoding
encoded = quote("Hello World! &?=+")
print(encoded) # Hello%20World%21%20%26%3F%3D%2B

# Form-style encoding (spaces as +)
encoded_plus = quote_plus("Hello World! &?=+")
print(encoded_plus) # Hello+World%21+%26%3F%3D%2B

# Decoding
decoded = unquote(encoded)
print(decoded) # Hello World! &?=+

PHP

PHP developers have a couple of options‚Äîurlencode() and rawurlencode()‚Äîdepending on whether you want spaces as + or %20:

// rawurlencode - uses %20 for spaces

echo $raw_encoded; // Hello%20World%21%20%26%3F%3D%2B

// Decoding

echo $decoded; // Hello World! &?=+

Best Practices

1. Encode Components Separately

Don‚Äôt just run encodeURI() on an entire link. Always encode individual parts like query parameters to avoid broken URLs.

2. Match Encoding to Context

Use percent-encoding for paths, encode both names and values in query strings, and don‚Äôt forget fragments if you‚Äôre working with them.

3. Handle Unicode Properly

When dealing with characters outside ASCII, always ensure you‚Äôre encoding with UTF-8 to avoid corruption.
